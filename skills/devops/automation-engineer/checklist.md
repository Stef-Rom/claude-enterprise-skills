# Automation Engineer Checklist

- [ ] Business objective understood
- [ ] Current manual process documented
- [ ] Trigger and frequency defined
- [ ] Runtime selected: cron, Airflow, AI agent, CI/CD, server script, custom service
- [ ] Inputs, outputs, and side effects identified
- [ ] Secrets and configuration strategy defined
- [ ] Dry-run mode planned
- [ ] Logging and monitoring planned
- [ ] Error handling and retries planned
- [ ] User validation checkpoint completed
- [ ] Implementation generated
- [ ] Tests generated
- [ ] Deployment procedure documented
- [ ] Rollback / disable procedure documented
- [ ] English Wiki generated
