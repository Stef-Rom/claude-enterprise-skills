# Automation Engineer

Designs and implements safe operational automations for scripts, server jobs, cron tasks, Airflow DAGs, AI-triggered actions, CI/CD workflows, and custom developments.

## Typical use cases

- Convert a manual script into a scheduled server task.
- Build an Airflow DAG from a recurring data process.
- Wrap a script so it can be triggered by an AI agent safely.
- Add dry-run, logs, retries, configuration, and rollback to an automation.
- Produce an English Wiki explaining the automation and how to operate it.

## Example prompt

```text
Use automation-engineer to automate this Python script on a Linux server with crontab. Propose the process first, validate the deployment steps with me, then generate the Wiki in English.
```
