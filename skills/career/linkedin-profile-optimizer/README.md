# linkedin-profile-optimizer

This Claude Skill updates and optimizes LinkedIn profiles for technical, cloud, DevOps, SaaS, AI-assisted development, and customer-facing engineering roles.

## Use it for

- Rewriting a LinkedIn headline.
- Rewriting the About section.
- Aligning LinkedIn with a CV.
- Adapting a profile to a target job offer.
- Improving recruiter search visibility.
- Building a stronger technical personal brand.

## Example prompt

```text
Use linkedin-profile-optimizer to update my LinkedIn profile for Senior Full Stack Python / React roles with Azure and AI-assisted development positioning.
```

## Install

Copy this folder into:

```text
~/.claude/skills/linkedin-profile-optimizer
```

or run the repository installer.
