# python-architect

This Claude Skill provides specialized guidance for: Use when designing, reviewing, refactoring, testing, or troubleshooting Python backend systems including FastAPI, Flask, Django, Pydantic, SQLAlchemy, MongoDB, Cosmos DB, async workers, Gunicorn, Airflow, and API services.

Install it by copying this folder into `~/.claude/skills/python-architect` or by running the repository installer.
