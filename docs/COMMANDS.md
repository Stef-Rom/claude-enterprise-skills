# Suggested natural commands

Claude Skills are triggered by descriptions, but these command-style prompts are convenient.

```text
/cv              Use cv-signature to improve my CV.
/job             Use job-match-analyzer for this offer.
/recruiter       Use recruiter-review to critique my CV.
/azure           Use azure-architect to review this architecture.
/cosmos          Use cosmosdb-expert to troubleshoot this database.
/devops          Use azure-devops-expert to create a pipeline.
/deploy          Use deployment-orchestrator to create a deployment plan.
/python          Use python-architect to review this backend.
/react           Use react-enterprise to review this frontend.
/ai-dev          Use ai-assisted-development to structure this AI coding workflow.
/saas            Use saas-architect to design multi-tenant SaaS architecture.
/docs            Use technical-writer to write documentation.
/customer        Use customer-solution-architect for this client requirement.
```


## Automation

```text
/automate <process or script>
```

Use the `automation-engineer` Skill to understand a process, propose an automation strategy, validate the deployment steps, implement the automation, and generate an English Wiki.

Examples:

```text
/automate this Python script with crontab and generate the English Wiki
/automate this data process as an Airflow DAG
/automate this operational action so it can be triggered safely by an AI agent
```
